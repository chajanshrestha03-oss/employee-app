# Employee Tracker App

A simple employee tracking application built with Python (Flask) and SQLite.

## Prerequisites

- Python 3.x

## Setup

1.  Navigate to the `server` directory:
    ```bash
    cd server
    ```

2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Running the App

1.  Start the server:
    ```bash
    python app.py
    ```

2.  Open your browser and go to `http://127.0.0.1:5000`.

## Features

-   Add new employees (Name, Role, Email).
-   List all employees.
-   Delete employees.
